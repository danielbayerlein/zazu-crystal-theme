# zazu-crystal-theme

> A crystal theme for [Zazu](https://github.com/tinytacoteam/zazu)

## Install

Add `danielbayerlein/zazu-crystal-theme` as value to `theme` inside of your `~/.zazurc.json` file.

```json
{
  "theme": "danielbayerlein/zazu-crystal-theme"
}
```
